# Favicon Images

Place your favicon images in this folder with the following names:

## Required Files:

1. **favicon-32x32.png** (32x32 pixels)
   - Standard favicon for most browsers
   - PNG format

2. **favicon-16x16.png** (16x16 pixels)
   - Smaller favicon for browser tabs
   - PNG format

3. **apple-touch-icon.png** (180x180 pixels)
   - Used when users add your game to iOS home screen
   - PNG format
